# Gaya Saya Web Source

## Uitleg

* Het configuratiebestand is ``config.toml``, dit is in TOML formaat en wijst zich hopelijk vanzelf.
* Alle ``icon`` velden in deze configuratie verwijzen naar iconen uit Font Awesome 5, zie https://fontawesome.com/icons
* Alle ``text`` velden in deze configuratie voldoen aan Markdown formaat: https://github.github.com/gfm/
* Alle ``source`` velden verwijzen naar fotobestanden in ``static/img/`` , waarbinnen voor elke categorie een subdirectory bestaat.
  Daarbinnen kan je je foto's uploaden. Let op, bestandsnamen zijn hoofldettergevoelig!
